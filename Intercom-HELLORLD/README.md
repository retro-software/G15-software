# Intercom HELLORLD

Intercom 1000-DP program by Cam Tabor (Discord @Thermal Dragon) to print the iconic Usagi Electric HELLORLD! (Hello World!) test output.

To run the program, load Intercom-1000DP into the emulator or G-15. Once in manual mode, enter the program into Intercom using one of three ways:
  1. Enter the program manually from the script in `HELLORLD-20251218-Entry-by-ThermalDragon.pdf` using Intercom command 500900.
  2. Enter the program using AutoHotKey from the script `BendixAutoHotkeyHexcodeScriptV1.3.txt`.
  3. Enter the program from the paper-tape image file `HELLORLD-20251218-Tape-Image.pti` using Intercom command 550952.

Then start the program using Intercom command 690932. The output should look like this:
```
 8  0 0000 0    8     00  000  8    000  0
 8  0 0    0    8    0  0 0  0 8    0  0 0
 8000 000  0    8    0  0 000  8    0  0 0
 8  0 0    0    8    0  0 0 0  8    0  0
 8  0 0000 0000 8000  00  0  0 8000 000  0
```

Files:

Original program as composed in a specialized spreadsheet.

AutoHotKey script for entering the program.

## HELLORLD-20251218-Entry-by-ThermalDragon.pdf

Script for entering the program manually from the emulator or G-15 keyboard.
